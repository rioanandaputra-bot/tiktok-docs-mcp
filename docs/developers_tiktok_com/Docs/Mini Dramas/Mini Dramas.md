# Mini Dramas Overview
TikTok Minis is a mobile webpage-based product framework that provides partners with an enriched web app ecosystem in the TikTok app. TikTok Minis offers seamless integration between mini apps from third-party services and the TikTok user experience, thereby extending user capabilities within the TikTok app. For example, TikTok Minis allows TikTok users to watch short dramas from third-party platforms within the TikTok app, as though they were viewing the content on the third party's own website.
In addition to allowing partners to tap into the TikTok ecosystem, TikTok Minis offers the following advantages:
- Accelerated loading capabilities and performance optimization for a better user experience
- Ability to systematically make web content available on TikTok without needing more complex tech integrations
- Customizeable configurations for transaction handling, advertising, and content protection
## Prerequisites
[[Before you start, you will need access to a TikTok developer account on the TikTok for Developers website](https://developers.tiktok.com/). You can sign up for an account](https://developers.tiktok.com/signup) if you don't have one.
